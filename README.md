# Catalog-Movie-ReactJS

Sebuah website yang bertema Catalog Movie yang dibuat <br> menggunakan ReactJs dan sudah terintegrasi Rest API

pre-request library :

- yarn add react-bootstrap bootstrap
- yarn add react-router-dom

API :

1. NewsAPI
2. UnplashAPI
3. TheMovieDB API
